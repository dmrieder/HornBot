Welcome, Dana and friends. The following web-based software program lets you track and record GPS data for the HornBot sculpture.

This program comprises four steps.
  1. Choose a title for the "performance" you are about to track
  2. Start tracking/recording gps-related data for your performance
  3. Pause tracking/recording gps-related data for your performance
  4. Return to the opening page to record another performance
